CSS Box Model

What is the CSS Box Model?

The CSS Box Model is the primary means of layout for everything on the web.
The purpose of the CSS Box Model is to define all elements as a box.

Properties of a CSS Box Model

  • Content

    • This area consists of content like text, image, or other types of media.

  • Padding

    • Padding is what adds space between the content and the edge of the box itself.

  • Border

    • It is the area between the boxâ€™s padding and margin. Its dimensions are given by the width and height of border.

  • Box sizing

    • The CSS box-sizing property is used to adjust or control the size of any element that accepts a width or height.
    It specifies how to calculate the total width and height of that element.

  • Margin

    • Margin is the property that gives space between elements. It can also be applied to the top, right, bottom, and left.
